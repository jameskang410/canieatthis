Why?

  • My wife is pregnant!
  • I got kinda tired of googling if she's allowed to eat something. Haha. Well, and I thought it could be a good chance to try some technology stuff I always heard about.
  • I missed web development. Except for CSS. I will never miss CSS. Which probably explains my "minimalist" (aka plain) websites. I regret nothing.

How?

  • I used BeautifulSoup to scrape a bunch of pregnancy websites my wife and I frequented.
  • The data was parsed into sentences and indexed as documents into Elasticsearch.
    • I added an integer field to every document to weigh documents' scores because there are definitely some documents that do not contain useful information. Users can help adjust this field (and therefore the search results) by using the Was this helpful? buttons next to each search result.
  • I used TextBlob to do sentiment analyses on the search results and return an overall conclusion (i.e. safe or not) based on the average of the sentiment scores. I'm not totally satisfied with this solution, but it's a start.
  • I used Redis to handle recent searches and build a list of missed searches that are e-mailed to me (and flushed) every night via cron jobs. Theoretically I'll start scraping other websites to cover these missed searches. Theoretically...
